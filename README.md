# 🎮 Mes projets Python : Jeux pour apprendre la programmation

Ce dépôt contient plusieurs petits jeux réalisés en Python dans le but d’apprendre et pratiquer le langage.  
L’objectif principal n’était pas seulement de créer des jeux, mais surtout de découvrir les bases du développement, la logique de programmation et l’organisation du code.

## 🚀 Pourquoi avoir créé ces jeux ?

J’ai choisi de développer des jeux car c’est une manière motivante et concrète d’apprendre Python.  
Chaque projet m’a permis de travailler différentes notions importantes :

- les variables et conditions
- les boucles
- les fonctions
- les classes et la programmation orientée objet
- les événements clavier et souris
- les bibliothèques graphiques (`tkinter`, `turtle`)
- la gestion de la logique de jeu
- l’aléatoire avec `random`

Créer des jeux permet aussi de voir immédiatement le résultat du code, ce qui rend l’apprentissage plus amusant et plus interactif.
